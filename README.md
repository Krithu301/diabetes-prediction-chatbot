
# Diabetes Prediction System with Chatbot

## Project Overview

The **Diabetes Prediction System with Chatbot** is a machine learning–based application designed to predict whether a person is likely to have diabetes based on medical parameters. The system also includes an **interactive chatbot** that helps users understand diabetes, provides guidance, and answers basic health-related questions.

This project aims to support **early detection of diabetes** and improve awareness through an easy-to-use interface.

---

## Features

* Predicts the possibility of diabetes using machine learning algorithms
* User-friendly interface for entering medical details
* Chatbot to answer diabetes-related queries
* Provides instant prediction results
* Helps create awareness about diabetes prevention and management

---

## Technologies Used

* **Programming Language:** Python
* **Machine Learning:** Scikit-learn
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask / Django
* **Dataset:** Pima Indians Diabetes Dataset
* **Libraries:** Pandas, NumPy, Matplotlib, Scikit-learn

---

## System Architecture

1. User enters health details (Glucose, Blood Pressure, BMI, Age, etc.).
2. The data is processed and sent to the **machine learning model**.
3. The model predicts whether the user is **diabetic or non-diabetic**.
4. The chatbot interacts with the user and provides diabetes-related information.

---

## Input Parameters

The model uses the following medical attributes:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin Level
* BMI (Body Mass Index)
* Diabetes Pedigree Function
* Age

## Installation

1. Clone the repository

```
git clone https://github.com/your-username/diabetes-prediction-chatbot.git
```

2. Navigate to the project folder

```
cd diabetes-prediction-chatbot
```

3. Install required libraries

```
pip install -r requirements.txt
```

4. Run the application

```
python app.py
```

5. Open the browser and go to

```
http://127.0.0.1:5000/
```

---

## Usage

1. Enter the required health parameters.
2. Click on **Predict**.
3. The system displays the prediction result.
4. Use the **chatbot** to ask questions related to diabetes symptoms, prevention, and treatment.
